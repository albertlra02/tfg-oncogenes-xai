# Datos de entrada

Los datos originales no se incluyen en este repositorio por tamaño y porque proceden de fuentes públicas externas.

Para reproducir el análisis, deben descargarse los ficheros indicados en el README principal del repositorio y colocarse en la carpeta `sources/`, que es la ruta utilizada por el notebook `TFG_Notebook.Rmd`.

La carpeta `datos/input/` se incluye únicamente para mantener la estructura solicitada del repositorio y documentar los datos necesarios para reproducir el trabajo.

## Ficheros necesarios

* `prad-rsem-fpkm-tcga-t.txt.gz`
* `luad-rsem-fpkm-tcga-t.txt.gz`
* `prostate-rsem-fpkm-gtex.txt.gz`
* `lung-rsem-fpkm-gtex.txt.gz`
* `NCG_cancerdrivers_annotation_supporting_evidence.tsv`

## Fuentes

* RNAseqDB: datos de expresión RNA-seq unificados de TCGA y GTEx.
* Network of Cancer Genes (NCG) 7.1: catálogo de genes de cáncer.
